# Laravel Design Rating

Star rating component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/rating
```


## Usage

### Livewire Component

```blade
<livewire:ld-rating />
```

### Blade Component

```blade
<x-ld-rating />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-rating-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-rating-views
```
